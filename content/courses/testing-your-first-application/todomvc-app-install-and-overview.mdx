# TodoMVC App Install & Overview

Aplikasi yang akan kami tulis tesnya dapat ditemukan [sini](https://github.com/cypress-io/cypress-realworld-testing-todomvc). Aplikasi ini adalah Aplikasi TodoMVC yang dibuat dengan React. Jangan khawatir jika Anda tidak tahu React. Anda masih dapat mengikuti meskipun Anda belum pernah menggunakan React sebelumnya.

## Clone Repo

Jika Anda telah menginstal git, Anda dapat mengkloning [TodoMVC repo](https://github.com/cypress-io/cypress-realworld-testing-todomvc) dengan memasukkan perintah berikut di terminal Anda:

```bash
git clone git@github.com:cypress-io/cypress-realworld-testing-todomvc.git
```

Alternatif untuk git adalah mengunduh file .zip dari repo dengan mengklik tombol hijau "Kode" dan kemudian "Unduh Zip."

![](/images/testing-your-first-application/todomvc-app-install-and-overview/Screen_Shot_2021-10-21_at_10.13.11-AM.png)

Selanjutnya, Anda perlu membuka repo di editor teks Anda. Kami akan menggunakan [VS Code](https://code.visualstudio.com/), tetapi jangan ragu untuk menggunakan editor teks apa pun yang Anda suka.

## Install

:::important

Node version 14 atau versi yang lebih baru.

:::

Proyek ini menggunakan [Yarn](https://yarnpkg.com/) untuk mengelola dependensi. Kalau kamu tidak punya [Yarn](https://yarnpkg.com/), anda dapat menginstalnya dengan perintah ini:

```bash
npm install -g yarn
```

Ubah direktori menjadi repo.

```bash
cd cypress-realworld-testing-todomvc
```

Instal dependensi dengan perintah berikut:

```bash
yarn install
```

![](/images/testing-your-first-application/todomvc-app-install-and-overview/Screen_Shot_2021-09-17_at_10.13.32_AM.png)

:::tip

Jika Anda kesulitan melihat tangkapan layar, Anda dapat mengekliknya untuk membuka gambar yang lebih besar.

:::

Memulai aplikasi:

```bash
yarn start
```

![](/images/testing-your-first-application/todomvc-app-install-and-overview/Screen_Shot_2021-09-17_at_10.13.44_AM.png)

Anda kemudian dapat membuka browser Anda ke URL server HTTP yang ditampilkan di konsol (e.g. **http://localhost:8888**), dan Anda akan melihat yang berikut ini:

![](/images/testing-your-first-application/todomvc-app-install-and-overview/Screen_Shot_2021-06-24_at_1.08.11_PM.png)

## App Overview

![](/images/testing-your-first-application/todomvc-app-install-and-overview/Screen_Shot_2021-06-24_at_1.14.12_PM.png)

Aplikasi ini adalah aplikasi todo dasar di mana Anda dapat menambahkan todos, todos lengkap, dan todos filter berdasarkan status aktif atau selesai mereka. Luangkan waktu untuk bermain-main dengan aplikasi untuk memahami fungsionalitas keseluruhan.

Pengujian adalah pola pikir, jadi sebelum kita dapat mulai menulis pengujian untuk aplikasi kita, pertama-tama kita perlu memahami apa yang perlu kita uji dan mengapa. Untuk mengetahui apa yang akan diuji, kita perlu memikirkan fungsionalitas inti aplikasi. Ajukan pertanyaan kepada diri Anda sendiri, "apa saja fitur terpenting dari aplikasi ini?" Mulailah dengan memecah fitur dan fungsionalitas aplikasi menjadi beberapa bagian. "Mengapa" akan menjadi lebih jelas setelah Anda memahami beberapa fungsi dan fitur inti.

Misalnya, karena ini adalah aplikasi todo, salah satu fitur terpenting adalah menambahkan todos. Jika pengguna tidak dapat menambahkan todos, seluruh aplikasi tidak berguna. Oleh karena itu, salah satu hal pertama yang harus kita uji adalah pengguna dapat menambahkan todos.

Dalam pelajaran berikutnya, kita akan belajar cara menginstal Cypress dan menulis tes pertama kita untuk memastikan bahwa kita dapat menambahkan todo ke aplikasi.
